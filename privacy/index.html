Privacy Policy

Effective date: July 25, 2025

This app ("Workout Timer") respects your privacy. Below we explain what data we collect, why, and how it's used.

1. Data We Collect

  • Anonymous usage data (via Firebase Analytics or App Store Analytics)
  • Workout data you create in the app (e.g. exercises, round durations)
  • Shared workout links (stored on a server if sharing feature is used)

2. How We Use Data

We use data only to:

  • Improve app performance and user experience
  • Allow workout sharing between users

3. Third-Party Services

This app may use the following third-party services:

  • Firebase Analytics – for anonymous usage statistics
  • Backend server – to store shared workout links

No personal data (like name, email, or location) is collected without your explicit action.

4. Your Rights

You may contact us at any time to:

  • Request deletion of shared data (e.g. workout links)
  • Ask about what data is stored and how it’s used
